SBURB Loader Screensaver by elrunethe2nd

Watch
Customization / Skins & Themes / Windows Utilities / Windows 7 Utilities / Other2011-2019 elrunethe2nd
Something I should have probably done a while ago.
Similar to the spirograph screensaver from ages ago.

===

Installation instructions in image.

MAKE SURE YOU PUT THE .scr FILE SOMEWHERE SAFE BEFORE YOU INSTALL.

You need it as long as you plan to have the screensaver. You can't shift it around after install.

===

Extended install instructions:
I'd suggest putting it somewhere safe, like C:/Windows/Resources or something. The thing I use to make these should generate it's own installers, but that feature still doesn't work.

You WILL need Adobe Flash Player installed. Chrome FP will NOT DO. Chrome comes packaged with it's own version to use which the rest of your system doesn't recognize. You will need the standard issue edition; download it with Firefox or IE.

By default, the settings have it set to Fit Screen. You can choose between having the flash fitted to screen, original size or stretch. Stretched looks bad, and Fit to Screen and Original both work.

===

To uninstall, simply delete the screensaver file.

===

Credits to Andrew Hussie for graphics. I threw it together.
