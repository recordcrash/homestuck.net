Homestuck Roleplaying Resources

Resources, tips and tricks for the avid Homestuck roleplayer, like vocabulary and acting guides, classified by character.

General Roleplaying Resources

The Roleplaying Sins

Not strictly a Homestuck guide, but we feel this is the most important roleplaying guide you'll ever read. Gives tips such as not forcing styles on others, keeping the RP clean, and so on.

Character Creation Guide: Homestuck Edition

A 2012 guide for people who haven't read Homestuck in a while and want a refresher on how to create an original character. Honestly if you need this you would benefit more from a reread, but who are we to judge.

Homestuck Troll Vocabulary Guides

Never use a human word for toiletry ever again! These guides will help you make your troll characters sound like trolls.

  • visionaryParacosmos' Troll Word Spreadsheet: A vocabulary table with categories and sources for each term, last updated in 2024.
  • Spiral's Canon Troll Vocabulary Sheet: A nearly exhaustive PDF dictionary of troll words, for both troll and human concepts. Updated for 2019.
  • Blueraspberry's Troll Vocabulary: This 2014 guide is not as complete as the former but adds some fanon words, as well as explaining some existing troll concepts that use human words (like "vast honk").
  • Fantrollreviews' Troll Vocabulary: The least complete vocabulary guide, and somewhat inaccurate, but features some troll idioms not in other lists (like "tripping globes").

Homestuck Pun Reference Guides

Three reference guides on how to use cat, horse and fish puns, useful for seadwellers, catdwellers and gross people.

  • Notes on Puns: A guide on how to make new puns sound natural instead of screeching dialogue to a halt.
  • Conglomeration Post of Puns: Contains most canon horse puns, cat puns and fish puns.
  • Fish Pun List 4 Rpers: The biggest resource for seadwellers, noting all the canon fish puns and even some idioms to use.

Homestuck Troll Culture Guides

Reference guides for troll biology and the troll calendar.

  • Alternian Calendar: A list to all the canonically used terms. See also this fanmade calendar with some fanon concepts.
  • Troll Biology, Et Al: All the canon about troll biology, with some trivia like teeth regeneration.
  • Further Troll Biology Guides: Guides that focus on troll mouths, bulges and eyes (this last one by Hussie).

Homestuck Dialogue per Character

A comprehensive (for the webcomic, anyway) list of every single line by each Homestuck character, organized and separated.

Homestuck Nickname Reference Guides

An exhaustive guide for the nicknames Homestuck characters use to reference each other. Worth clicking on every link in the post for further nickname resources, like the ones Sollux uses, or the fanon ones that aren't in the comic.
